# 16x2  LCD Display
A basic Arduino project that initializes a 16x2 LCD and displays a custom message. Demonstrates how to connect and use the LiquidCrystal library for text output.

## Components
- 1x Arduino UNO
- 1x LCD 16x2
- 1x 220Î© Resistor
- 1x Potentiometer
- Jumper Wires
- Breadboard
