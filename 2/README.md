# Sequential LED Blinker

This code turns on each LED (green, yellow, red) one by one for 1 second, then turns it off before moving to the next. The sequence repeats in a loop.

## Components
- 1x Arduino UNO  
- 3x LEDs (green, yellow, red)  
- 3x 220Î© Resistors  
- Jumper Wires  
- Breadboard
