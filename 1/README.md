# Blink LED

This Arduino code blinks three LEDs (green, yellow, red) on pins 13, 12, and 11. All LEDs turn on and off together every 1.5 seconds.

## Components
- 1x Arduino UNO
- 3x LEDs (green, yellow, red)
- 3x 220Î© Resistors
- Jumper Wires
- Breadboard
