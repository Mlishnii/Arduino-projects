# Arduino Melody Player
Plays a predefined sequence of musical notes using a buzzer. Utilizes pitches.h for note frequencies and the tone function for sound generation.

## Components
- 1x Arduino UNO
- 1x piezoelectric buzzer
- Jumper Wires
- Breadboard
