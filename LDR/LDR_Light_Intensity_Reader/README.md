#  LDR Sensor Reading
Reads light intensity from an LDR on A0 and prints the value to the serial monitor every 500 ms.

## Components
- 1x Arduino UNO
- 1x Photoresistor (LDR)
- 1x 10kÎ© Resistor (for LDR voltage devider)
- Jumper Wires
- Breadboard
